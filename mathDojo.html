Ardor® Math Dojo

The Ardor® Math Dojo is a mobile Algebra training application. Students will use MathDojo to work on small, randomly generated problems, that will ramp up in difficulty as the student becomes more proficient. The application will communicate with a remote web-server using a REST API to report on their progress. This progress data will be used to generate reports for interested stakeholders such as the students, parents, and educators.
